# Warehouser

An inventory handeling application for handeling what is in stock what has to be bought and more.
Compiled for Windows, OS X(Mac), Android and IOS. Using .NET MAUI with the option to use a centralized web api for multiple users or a local approach for a single user.

Written in C# .NET 10.0

Utilities for:
- Printing "to buy" lists.
- Managing inventory.

The web api has port defaulted to **8426 for http and 8427 for https**.
